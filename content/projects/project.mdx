A simple Reactjs TodoList App

Preview: [https://todoapp-one-sigma.vercel.app/](https://todoapp-one-sigma.vercel.app/)

## Prerequisites

Before running this project, make sure you have the following installed on your machine:

* Node.js ([https://nodejs.org](https://nodejs.org/))

## Technologies Used

* Reactjs
* TailwindCSS

## Getting Started

1\. Clone repository

```shell
git clone[repository URL]
```

2\. Open project directory

```shell
open folder directory
```

3\. Install all dependencies

```shell
npm i
Run app
npm run dev

```

1. Open your browser and visit [http://localhost:5173](http://localhost:5173/) to see the application running.

Thank you for making it to the end!
